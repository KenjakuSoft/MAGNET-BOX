Use with download managers (JDownloader, IDM, VLC…)

      
Download & stream links also accept the token as HTTP Basic auth, so a plain URL works in any external app — just put your token in the link:

      
https://YOUR_USERNAME:YOUR_TOKEN@HOST/download/3/0

      
In the app, use a file's ⧉ Copy link button — it copies a ready-to-paste link with your key already embedded. Anyone with that link can download as you, so treat it like a password; rotate the token to revoke old links.

    


    

      
API quickstart

      
All endpoints accept the bearer token instead of a login cookie. Full reference: /docs · openapi.json

      
# list torrents
curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://HOST/api/torrents

# add a magnet or direct link
curl -H "Authorization: Bearer YOUR_TOKEN" -H "content-type: application/json" \
  -d '{"source":"magnet:?xt=..."}' http://HOST/api/add
