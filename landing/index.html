Everything in the box

A complete download & stream server — not a toy.

Built like a real product: a slick dashboard, multi-user access control, an admin console, and an API. You host it; it does the rest.

🧲

Magnets, torrents & direct links

Paste a magnet, a .torrent URL, or any http(s) link — it's auto-detected, downloaded at full speed to your server, and turned into a private direct link.

▶️

Stream while it downloads

An in-app player streams video and audio with seeking before the download even finishes, and auto-loads subtitle tracks from the torrent.

👥

Multi-user, invite-only

Closed, invite-only registration with invite codes you hand out during an open window.

🔐

Real security

argon2 logins, TOTP 2FA with recovery codes, sessions, SSRF protection, security headers.

🛠️

Admin console

Live CPU/RAM/disk, per-user bandwidth, audit log, sessions, bulk controls — full control.

🗑️

Auto-expiry

Optionally delete content after N days to reclaim disk automatically. Keeps the box lean.

📘

OpenAPI + tokens

Per-user API tokens and an OpenAPI 3.1 spec, so your own scripts and media tools integrate cleanly.

⚡

Limits & throttles

Global speed caps and per-user concurrency limits keep the server healthy.

🎬

Play any format

With ffmpeg, MKV / H.265 / AVI and more transcode on the fly to play in any browser — even while still downloading.

📡

RSS auto-download

Subscribe to feeds with a keyword filter and new matching torrents grab themselves. Set it and forget it.

🔔

Completion alerts

Get pinged on Discord, ntfy or any webhook the moment a download finishes.

📱

Installable app

Add it to your phone's home screen and run it in a standalone window, with a light / dark theme.

📥

Quick-add bookmarklet

One click on any torrent page sends the magnet straight to your box — no copy-pasting.

🚀

Guided setup

First run opens a wizard to create your own account — no random passwords to dig out of a console.

Up in minutes

Three steps to your own box.

Run the binary

Drop the single binary on any Linux server and start it behind Caddy for automatic HTTPS. Systemd unit included.

Sign in & lock it down

Log in, change the admin password, turn on 2FA, and keep registration invite-only.

Add a link

Paste a magnet or URL. Watch it download, then stream it in-app or pull the direct link — from any device.

Questions

Good to know.

Is it really free?

Yes. MagnetBox is free, self-hostable software — no license key, no subscription. Donations are optional and simply fund continued development. You get the single binary plus deploy files and docs; you run it on your own server and we never see your data.

What kind of tool is it?

A neutral, general-purpose self-hosted download & streaming tool — like qBittorrent or Plex. You run your own private instance on your own server and use it however you like.

What do I need to run it?

Any small Linux VPS (≈2 vCPU / 4 GB RAM) with disk for your downloads and a domain. A starter box runs a few dollars a month. The README has a complete beginner deploy guide.

Does it phone home or need our servers?

No. It runs entirely on your machine. There's no telemetry and no dependency on us to keep working.

Can other apps use it?

Yes — there's an OpenAPI 3.1 spec and per-user API tokens, so your scripts and media tools can drive it programmatically.

How can I support the project?

Donations (Ko-fi, PayPal, or crypto) are the main way — they fund maintenance, security fixes and new features. Sharing the project and reporting bugs helps just as much.
